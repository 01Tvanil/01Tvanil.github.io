Git提交时报Everything up-to-date的解决方法

问题描述

当gitbash中输入git push -u orgin master的时候，git给出Everything up-to-date，然而库中并未成功上传

问题解决方法

先创建一个新的分支提交改动

git branch newbranch

切换工作分支

git checkout newbranch

将改动提交到新分支

git add .

git commit -a

返回到主分支，并将新分支提交的改动合并到主分支

git checkout master

git merge newbranch

然后再进行一次push操作

git push -u origin master

问题解决，如果不行，可能切换分支的时候出了问题，每次切换可以用git branch查看，带*号的为主分支

原理

push操作并不会将所有分支push掉
